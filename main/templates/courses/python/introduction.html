What is Python?

Python is a popular programming language created by Guido van Rossum and released in 1991. It is extensively used for:

  • Web development (server-side)
  • Software development
  • Mathematics
  • System scripting

What can Python do?

  • Python can be used on a server to create web applications.
  • Python can be used alongside software to create workflows.
  • Python can connect to database systems, read and modify files.
  • Python can handle big data and perform complex mathematics.
  • Python is used for rapid prototyping or production-ready software development.

Why Python?

  • Python works on different platforms (Windows, Mac, Linux, Raspberry Pi, etc).
  • Python has a simple syntax similar to the English language.
  • Python allows developers to write programs with fewer lines compared to other languages.
  • Python runs on an interpreter system, enabling quick prototyping.
  • Python supports procedural, object-oriented, or functional programming.

Good to know

  • The most recent major version of Python is Python 3, which will be used in this tutorial.
  • Python 2, although not updated, is still quite popular.
  • In this tutorial, Python will be written in a text editor, though IDEs like Thonny, PyCharm, Netbeans, or Eclipse are also useful.

Python Syntax compared to other programming languages

  • Python was designed for readability, resembling the English language with mathematical influence.
  • Python uses new lines to complete a command instead of semicolons or parentheses.
  • Python relies on indentation (whitespace) to define scope, unlike other languages using curly-brackets for scope.
